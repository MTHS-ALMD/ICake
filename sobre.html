Objetivo

  • O objetivo deste projeto é disponibilizar uma ferramenta, um site, que possibilite conectar microempreendedores, vendedores e boleiras.
  • O iCake disponibiliza uma área para que o vendedor consiga mostrar seus produtos para o consumidor.
  • Além de contar com um sistema de classificação de vendedores, gerando uma concorrência entre eles, para que entreguem o melhor produto.
